Web

Mappin

Web

Mappin is a Full Stack travel log application that allows a user to create an account, post a pin on the map by creating a post using a title, description, rating and images. You can also view all other pins on the map posted by yourself and other users and search pins by specific location as well. The front-end was developed with React.js and Mapbox, while the backend was developed with Node.js, Express.js, and MongoDB Atlas.
